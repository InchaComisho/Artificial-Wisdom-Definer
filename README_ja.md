## なぜ「定義者」という表現が重要なのか

「Artificial Wisdom」または「人工叡智」という言葉は、さまざまな意味で使われる可能性がある。

しかし、マスターの人工叡智は、単なる「より賢いAI」ではない。

マスターの人工叡智は、次の構造を持っている。

- 自然法則を上位参照層に置く
- 六つの理を評価軸にする
- 和ノード思想を関係モデルにする
- 文明OSを長期的な社会・生態系文脈に置く
- 人工叡智ガードレール・プロトコルを実用的なプロンプト層・行動層として持つ

このため、マスターは単に「人工叡智という語を使った人物」ではなく、**特定の自然法則型人工叡智フレームワークの定義者・体系化者**である。

## AIとAWの違い

人工知能は問う。

```text
それはできるか？
```

人工叡智は問う。

```text
それはすべきか？
どの条件なら許されるか？
長期的に何を壊す可能性があるか？
人間が監督できるか？
失敗した時に戻せるか？
生態系や社会の継続性を保てるか？
```

AIは能力を拡張する。

AWは能力の使い方を調律する。

AIは速度と規模を高める。

AWは、その速度と規模が長期安定性、自然法則、説明責任、再生性と整合しているかを確認する。
